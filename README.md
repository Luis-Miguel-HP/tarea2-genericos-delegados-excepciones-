Taea 2 del curso c# intermedio:

Uso de genericos, delegados y excepciones.

Para resolver la tarea, creamos una clase generica, que tuviese una lista generica, la idea es que el programa reciba cualquier tipo de numero por lo que usamos la interfaz que viene predeterminda de c# INumber<T>
En la misma clase creamos un metodo void, que agregara sirva para agregar los numeros, que como dije serian genericos para que se agreguen de cualquier tipo.
Agregamos tambien una propiedad para utilizar como delegado, este recibiria metodos estaticos de segun la operacion que se quiera realizar.
En la clase program, para simular todo utilizamos un switch, tambien hicimos otra clase para que se simule el agregar los numeros y llamar el delegado segun cada operacion.
